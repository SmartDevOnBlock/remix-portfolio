# Remix-portfolio

The Remix Ethereum IDE themed developer portfolio website built with Next.js and deployed on Vercel.

![remix-portfolio banner](https://imgur.com/at9FSKK.png)

For other features and themes suggestions, please contact me.

## Environment Variables

For fetching your articles from dev.to, create an `.env.local` file inside the project directory. Check the `.env.local.example` file for more information.

## Running Development Server

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

All Remix related components can be found in the `components` folder. To change the content of the portfolio, check out the `pages` folder. To add or remove pages, modify `components/Sidebar.jsx` and `components/Tabsbar.jsx`.
